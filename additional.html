Revisits

A guide on revisit processess

DMB Support team also handles visits that were done by the Deployment team, and were incomplete and/or completed with issues, for an example:
-4 out of 6 Displays replaced on first visit
-1 out of 4 Displays Dead on Arrival/Out of Box failure (DOA/OBF)
-Display replaced, cables non-functional (Could be any hardware and/or cables)
-Displays installed into wrong positions

The visit should be investigated, to determine the outcome, and apply appropriate solutions, to have the remaining issues solved.
Here is a guide on how to handle different processess:

Not all Displays replaced


Check if all of the hardware was sent and deliverd through the Samsung B2B portal,
and check the excel base Michael Sei Tejpal is sending every day, where all of the shipped Samsung screens are located.
If the screens are Stratacaches AUO/LG screens, it can be checked with the FedEx tracking and/or Day 2 Shipped Smartsheet.
If delivery was completed, schedule technicians to replace the screen.


Dead on Arrival/Out of Box failure hardware:

If the screen is a Samsung screen, a replacement unit can be ordered through the Samsung B2B portal, track and schedule technicians when delivered, while updating the snow with the current state of the ticket.
If the screen is an AUO/LG unit, or a Media Player or cables, a gratis order should be sent to the sales team, if technicians didnt take all of the pictures needed where Stratacache approved to send a replacement unit.


Display replaced, other hardware non-functional:

If, for an example, the Display was replaced, and it was identified that the HDMI cable is also an issue, a new Sales request should be made for the hardware.


Display installed into wrong positions:

It should be identified where the screens where the screens were installed, and technicians should be scheduled to swap and install Displays into correct enclosures.
If the Displays are Stratacache AUO screens, it should be noted that old LG and new AUO screens shouldnt be in the same enclosure.


Enclosure issues:

If the enclosure has issues preventing the installation from taking place, the issues should be dealt with, if possible.
For stripped screws, technicians should be scheduled and informed beforehand that they should bring tools needed to remove stripped screws.
For any clearance issues, preventing the enclososure to be fully opened due to something blocking the path, the store should be informed to try and resolve that issue.
