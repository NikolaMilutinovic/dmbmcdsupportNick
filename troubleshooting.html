General troubleshooting

A generic troubleshooting process for a non-functional screen

Further troubleshooting is required to determine the cause of failure.
Check for any physical damage.
If there is physical damage, please note that with any pictures Check to see if screen has power.
If not, double check the power outlet and power cable.
Use Samsung remote to check menu/power (Screen power can be checked if the infrared sensor light is lit when the screen is powered off and connected to power).
If screen powers on with remote, use remote to check if the Menu appears and is visible.
Recommended step: Turn the ODMBs breaker off for a couple of minutes and then back on.
Use a working HDMI cable from another screen or board to see if that fixes the issue.
If yes, then the store needs to order a new HDMI cable.

Screen power and HDMI

A generic troubleshooting process the screens power and hdmi issues

If screen powers on with remote, use remote to check if the Menu appears and is visible.
Recommended step: Turn the ODMBs breaker off for a couple of minutes and then back on.
Use a working HDMI cable from another screen or board to see if that fixes the issue.
If yes, then the store needs to order a new HDMI cable.
Call Stratacache to see if the media players are online and if they can see the screens.
If the above steps do not fix the issue, the screen is power faulty and needs replacement under warranty.


Bright/white spots on screen

A generic process for a screen with white spots screen

It's most likely that the GLUE on the Backlight LED Defusers has weaked, and they have fallen off. The screen needs to be replaced.


Dim screen

A generic process for a dim screen

Backlights/LED failing, and need replacement.


Flickering screen

A generic process for a flickering screen

Failure on the power board, or a lose connection between the Power board + backlights, or between the strips themselves. The screen needs to be replaced.


No backlight at all

A generic process for a a screen with no backlights

Turn on screen, and turn on flashlight on the phone, if the pictures are visible, it's a backlight failure, and need replacement.


Stripped screw/h2>

A generic process for stripped screws

Best way to handle this issue is to schedule a revisit, and inform the technicians to bring tools required for a stripped screw situation beforehand.
Remove the side panels, top panel, and bottom panel.
Remove the screen that does not have the rusted screw from the enclosure.
Keep the all 4 bolts in the screen with the rusted screws attached to keep screen secure to the frame.
Remove the 4 screws holding the bottom bracket to the post.
Support the bottom bracket which should be able to move with only the two bolts attached from the screen with the rusted screw.
Carefully loosen the top two bolts on the screen while supporting the bottom bracket and screen. Once the bolts are loosened but not removed, lift up and pull the screens out.
Alternatively the top two bolts could be completely removed if determined to be easier by the tech.


Stratacache-Hooks dont align

A generic process for a an enclosure where hooks dont align

The base of the bracket on the TV is too wide for the pintle in the case.
We just loosened the four bolts on the bracket on the TV and squeezed it together, then tightened the bolts.
This made the brackets narrower and allowed the TV to slide into place in the case

RS232 cable

A generic process for RS232 cable troubleshooting

⦁ Swap RS-232 cables for effected screens with known working screen RS-232 cables.
If issue is resolved/ images return, escalate to NCR for cable replacement.
If issue persists, move to media player troubleshooting step (next step.)
⦁ Swap media player of effected screen with known working media player
If issue is resolved/ images return, escalate to Stratacache for media player replacement.
If issue persists, proceed to power cable troubleshooting (next step.)
⦁ Swap power cable from known working media player to the effected media player.
If issue persists, escalate to Coates for a warranty screen replacement.
If issue is resolved, escalate to NCR for media player power cable replacement.


HDMI cable

A generic process for HDMI cable troubleshooting

⦁ Swap HDMI cables for effected screen(s) with known working screen HDMI cables.
(The known working cable can be used from a different media player connection.)
If alternative HDMI cable resolves issue/images return, escalate to NCR for cable replacement.
